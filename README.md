# InfoManager

Information Manager

## 功能实现

* 登陆/注册
* 密码修改
* 信息处理
  - 添加信息
  - 删除信息
  - 查询信息
  - 修改信息

## 数据库结构

* 用户数据

```json
{
  "uid": 1000000,
  "name": "evi0s",
  "passwd": "asdibbdbiubquibdkqb",
}
```

* 信息数据

```json
{
  "year": 2018,
  "month": 10,
  "day": 14,
  "source": "公开电话",
  "category": "规划建设",
  "subcategory": "asd",
  "title": "balabalabala",
  "subtitle": "balabalabala",
  "content": "rua",
  "remark": "ruarua"
}
```
